Project

A steel project is done only once, whereas work during the operation of a steel plant is of repetitive nature, and management of one-time jobs is different from managing repetitive types of jobs. Also, the people who work on a steel project may be reassigned to other jobs once the project is completed, so the team is temporary.

During the project management, specific tasks and resources are required to be managed against a time-oriented set of objectives.

Planning

Project planning is done to increase the likelihood that the project will be implemented efficiently, effectively and successfully. Also, it is very much necessary that required time, resources, and efforts are provided as well used for the preparation of the project plan since a defective plan can lead the project to a situation which becomes out of control of the project team.

A project plan is a formal, approved document of the project which guides the project execution and is required for the management and control of the project.
